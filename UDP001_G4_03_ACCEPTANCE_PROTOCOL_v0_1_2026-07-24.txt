4. COUNT LINE
======================================================================

N-Uu-Kk-Ff-Ii-Bb-Ee-Hh-Vv-Zz-Qq

u = input/output update rows
k = policy/selected signal count
f = FLOAT row count
i = INTEGER row count
b = BOOLEAN row count
e = ENUM row count
h = BITFIELD row count
v = valid update count
z = invalid update count
q = dtype conversion failure count
