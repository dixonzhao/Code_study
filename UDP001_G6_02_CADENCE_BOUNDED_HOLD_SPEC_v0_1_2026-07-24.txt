- S__updated and S__update_count as current-frame provenance;
- input file unchanged.

NONE:

- copy all S columns exactly from Gate 5.

HOLD_LAST_BOUNDED:

Current frame has a valid real update:

- preserve current value/label;
- updated/update_count remain current-frame values;
- age=0;
- valid=1;
- preserve VALID_UPDATE/VALID_AGGREGATED reason;
- record this frame as the last valid real update.

Current frame has an invalid real update:

- preserve the invalid current result;
- do not carry an older value over it;
- set a hold barrier;
- future no-update frames remain invalid until a new valid real update.

Current frame has no update and an unblocked prior valid real update:

- age = current FrameCount - last valid real-update FrameCount;
- if age <= MaxAgeFrames:
  carry value/label;
  updated=0;
  update_count=0;
  valid=1;
  missing_reason=VALID_CARRIED;
- if age > MaxAgeFrames:
  value/label blank;
  updated=0;
  update_count=0;
  valid=0;
  missing_reason=STALE;
  retain numeric age for audit.

Current frame has no usable prior update:

- value/label blank;
- updated=0;
- update_count=0;
- age blank;
- valid=0;
- missing_reason=NO_UPDATE or the controlled barrier state.

Never carry EVENT_FLAG.
Never convert blank/stale/invalid to zero.
Never use a carried value as a new real-update origin.
