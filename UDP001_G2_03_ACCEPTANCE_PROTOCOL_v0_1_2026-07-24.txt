2. SHARED CODE
======================================================================

======================================================================
3. B GROUPS
======================================================================

B1 — Code and self-test

Position 1 = all 12 self-tests passed
Position 2 = foundation imported successfully
Position 3 = module import caused no execution/GUI side effect
Position 4 = source program and foundation remained unchanged

B2 — Selection

Position 1 = selection template created
Position 2 = selected-signal count is nonzero
Position 3 = every Required=1 signal resolved
Position 4 = CanonicalName values are unique

B3 — Update Stream semantics

Position 1 = output is non-empty
Position 2 = SourceRowIndex is monotonic and never reordered
Position 3 = no forward fill or blank-to-zero operation occurred
Position 4 = RawCode and PhysicalValue are separate and unselected output count
             is zero

B4 — Scale and safety

Position 1 = pandas chunksize path executed
Position 2 = max-source-row limit respected
Position 3 = incomplete/invalid update states are counted explicitly
Position 4 = exactly three Gate 2 outputs were created and input remained
             read-only

For B4 position 3:

2 means the counting mechanism was verified, including when the real count is
zero.

======================================================================
4. COUNT LINE
======================================================================

N-Rr-Mm-Ss-Pp-Ee-Kk

r = source rows scanned
m = decoded message updates
s = decoded signal-update rows
p = incomplete-message count
e = invalid-byte/error count
k = selected signal count

Example format only:

N-R100000-M5200-S10400-P3-E0-K6

Do not reuse the example numbers.

======================================================================
5. RESULT CODE
======================================================================

C-0 = PASS
C-1 = generated source incomplete or syntax failure
C-2 = self-test failure
C-3 = foundation import/version failure
C-4 = selection creation or resolution failure
C-5 = Raw source/DBC read failure
C-6 = empty output or no decoded updates
C-7 = ordering/fill/unselected-output semantic failure
C-8 = row-limit/chunk/input-safety/output-file failure
C-9 = other blocking failure; add one Memo

PASS requires:

- B1-2222
- B2-2222
- B3-2222
- B4-2222
- C-0
