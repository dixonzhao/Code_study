======================================================================
3. B GROUPS
======================================================================

B1 — Code and self-test

Position 1 = all 14 self-tests passed
Position 2 = Gate 1/Gate 2 modules remained import-safe and unchanged
Position 3 = Gate 3 import caused no execution/GUI side effect
Position 4 = generated source is one complete Python file

B2 — Authoritative FrameCount

Position 1 = configured CanonicalName resolved uniquely
Position 2 = observed FrameCount values are numeric integers
Position 3 = FrameCount never decreases in SourceRowIndex order
Position 4 = same-SourceRowIndex frame observations do not conflict

B3 — Assignment semantics

Position 1 = exact and within-limit assignment statuses are explicit
Position 2 = before-first and limit-exceeded rows remain unassigned
Position 3 = FrameAssignmentAgeRows is correct
Position 4 = no measurement value was filled or changed

B4 — Scale and safety

Position 1 = pandas chunksize path executed with cross-chunk state
Position 2 = output row count and SourceRowIndex order equal input
Position 3 = assignment limit was respected
Position 4 = exactly three Gate 3 outputs were created and input remained
             read-only

======================================================================
4. COUNT LINE
======================================================================

N-Uu-Rr-Ff-Dd-Aa-Ll-Qq-Xx

u = input/output update rows
r = unique SourceRowIndex count
f = authoritative FrameCount observation count
d = distinct FrameCount count
a = total successfully assigned update rows
l = unassigned-limit update rows
q = assignment-age q95 in source rows, rounded to at most three decimals
x = FrameCount decrease count

Example format only:

N-U500000-R90000-F4000-D3900-A499500-L300-Q12-X0

Do not reuse the example numbers.

Rows before the first observed FrameCount are not included in `l`.

======================================================================
5. RESULT CODE
======================================================================

C-0 = PASS
C-1 = generated source incomplete or syntax failure
C-2 = self-test failure
C-3 = Gate 1/Gate 2 import or schema failure
C-4 = authoritative FrameCount resolution failure
C-5 = non-numeric/fractional/decreasing/conflicting FrameCount
C-6 = empty input or empty output
C-7 = assignment/age/measurement-mutation semantic failure
C-8 = chunk/order/row-count/input-safety/output-file failure
C-9 = other blocking failure; add one Memo

PASS requires:

- B1-2222
- B2-2222
- B3-2222
- B4-2222
- C-0

`L` may be greater than zero without failing Gate 3 when the rows are correctly
marked FRAME_UNASSIGNED_LIMIT. Production assignment-limit approval is a later
decision.
