CONFIRMED OPERATING CONTEXT

1. Framecount is the only currently trusted key for synchronization with the
   Stereo Camera video.
2. DateTime and Time[us] are not trusted for this dataset.
3. P003 is used for most normal runs before P004.
4. P001 is only a signal-discovery tool for unfamiliar tasks and is outside
   this review.
5. Different signals may use different invalid/default values.
6. A numeric zero can be a real value for some signals, so zero must not be
   assumed to have one universal meaning.


IMPORTANT DISTINCTION TO AUDIT

Keep these four states separate in the report:

A. NO_UPDATE
The signal was not transmitted or updated on the current source row/frame.

B. INVALID_SENTINEL
The source signal contains a signal-specific invalid/default numeric code.

C. TRUE_ZERO
The source measurement is valid and its physical value is zero.

D. IMPUTED_VALUE
The value was created by forward fill, zero fill, or another software rule.

Do not assume that a blank cell means INVALID_SENTINEL. Determine from code
what created the blank and whether that meaning remains recoverable.
